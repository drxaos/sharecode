# sharecode

Сервис для совместного редактирования кода.
Будет использоваться для проведения собеседований.

Основной сценарий использования:

- Пользователь заходит в корень сайта и видит страницу с логотипом и кнопкой "Создать комнату"
- Когда пользователь нажимает кнопку "Создать комнату", сервис генерирует уникальный id комнаты и перенаправляет браузер по адресу /room/<id>
- Когда пользователь переходит по адресу комнаты, сервис отображает страницу комнаты с редактором кода
- Пользователь может скопировать адрес комнаты (url) и передать другим пользователям, чтобы они тоже подключились к этой комнате
- При изменении кода на странице комнаты эти изменения видят все пользователи, зашедшие на страницу этой комнаты
- Также в редакторе кода видно где стоит курсор каждого пользователя (каждый своим цветом)
- Если пользователь выделил часть текста, то у всех она подсвечивается цветом этого пользователя

Требования:

- Бэкенд на golang
- Фронтэнд на js+react
- БД не нужна, данные хранятся в памяти
- Для получения содержимого комнаты фронтенд соединяется с бэкендом через вебсокет
- На странице комнаты есть выбор подсветки синтаксиса (plain, java, js, ts, golang, c# и т.д.), применяется для всех в комнате
- На странице комнаты есть выбор размера шрифта, применяется только для текущего окна
- Слева у редактора кода отображаются номера строк
- Комната уничтожается не раньше чем через 5 минут после отключения последнего пользователя

Используй популярные библиотеки для компонентов приложения.
